## 📋 PASSO A PASSO

### **PASSO 1: Abrir Google Drive**
1. Acesse seu Google Drive
2. Localize a pasta com documentos do SKINBELLA

### **PASSO 2: Para Cada Documento**

#### **2.1 Identificar o Tipo**
Pergunte: "Este documento é sobre o quê?"
- Copy (VSL, email, landing page, quiz, ad)?
- Produto (briefing, substâncias, diferencial)?
- Público-alvo (persona, dores, objeções)?
- Branding (tom de voz, palavras)?

#### **2.2 Escolher a Pasta Correta**

| Tipo de Documento | Pasta de Destino |
|-------------------|------------------|
| **Script de VSL** | `copy/vsl/` |
| **Copy de landing page** | `copy/landing-pages/` |
| **Email marketing** | `copy/emails/` |
| **Copy do quiz** | `copy/quiz/` |
| **Anúncio Facebook/Insta** | `copy/ads-social/` |
| **Briefing de produto** | `produtos/briefings/` |
| **Lista de ingredientes** | `produtos/substancias/` |
| **Diferenciais vs mercado** | `produtos/diferenciais/` |
| **Persona/público** | `publico-alvo/` |
| **Tom de voz** | `branding/` |

#### **2.3 Copiar Conteúdo**
1. Abra o documento no Google Drive
2. **Selecione tudo** (Cmd+A)
3. **Copie** (Cmd+C)

#### **2.4 Criar Arquivo Aqui**
1. No Mac, abra Finder
2. Navegue até: `/Users/ayan/skinbella-app/docs/referencias/[pasta-correta]/`
3. Crie novo arquivo: `nome-descritivo.md`
4. Abra com TextEdit ou qualquer editor
5. **Cole** o conteúdo (Cmd+V)
6. **Salve**

#### **2.5 Nomear Bem**

✅ **BONS NOMES:**

## 🚀 ATALHO RÁPIDO (Se tiver MUITOS docs)

Se você tem 20+ documentos no Drive:

### **Opção A: Download em Massa**
1. No Drive, selecione todos os docs da pasta SKINBELLA
2. Clique com direito → **Download**
3. Drive vai baixar um .zip
4. Extraia o .zip
5. Mova cada arquivo para a pasta correta aqui

### **Opção B: Compartilhar Link**
1. Você me compartilha o link da pasta do Drive (view only)
2. Eu vejo a estrutura
3. Te digo exatamente onde colocar cada um
4. Você copia/cola
