## 📝 O QUE COLOCAR AQUI

### **persona.md** - Persona Principal
Descrição detalhada da cliente ideal.

**Estrutura sugerida:**
```markdown
# Persona SKINBELLA - Maria (nome fictício)

## Dados Demográficos
- **Idade:** 25-45 anos
- **Gênero:** Feminino
- **Localização:** Brasil (urbana)
- **Renda:** R$ X - Y/mês
- **Profissão:** [tipo de trabalho]
- **Estado civil:** [casada, solteira, etc.]

## Perfil Psicográfico
- **Preocupações principais:** Envelhecimento, manchas, rugas
- **Nível de conhecimento skincare:** Intermediário
- **Tempo disponível:** Pouco (quer rotina fácil)
- **Investimento skincare atual:** R$ X/mês

## Dores (Problemas)
1. Pele com sinais de envelhecimento precoce
2. Não sabe qual rotina seguir
3. Já tentou vários produtos sem resultado
4. Falta de tempo para skincare elaborado
5. Confusão com tantas informações na internet

## Desejos (Objetivos)
1. Pele de porcelana (sem manchas, lisa, jovem)
2. Rotina simples e eficaz
3. Reverter anos de envelhecimento
4. Sentir-se bonita e confiante
5. Método cientificamente comprovado

## Objeções Comuns
1. "Será que funciona mesmo?"
2. "É muito caro?"
3. "Vou ter que usar muitos produtos?"
4. "Tenho pele sensível, vai irritar?"
5. "Quanto tempo leva para ver resultados?"

## Onde ela está (Online)
- Instagram (conteúdo de beleza)
- Pinterest (inspirações)
- TikTok (tutoriais rápidos)
- Grupos de Facebook (skincare)
- YouTube (reviews de produtos)

## Influências
- Dermatologistas
- Influenciadoras de beleza
- Amigas com pele bonita
- Antes/depois reais
- Estudos científicos
```

### **objecoes.md** - Objeções e Respostas
Todas as objeções que a cliente pode ter + como responder.

**Exemplo:**
```markdown
# Objeções Comuns e Respostas

## 1. "É muito caro"
**Resposta:**
- Comparar com tratamentos em clínica (R$ 500-2000)
- Dividir preço mensal (R$ 19 = menos de R$ 1/dia)
- Mostrar economia vs comprar produtos separados

## 2. "Não vai funcionar pra mim"
**Resposta:**
- Garantia de 7 dias
- Depoimentos de mulheres com pele similar
- Protocolo personalizado
```

### **jornada-cliente.md** - Mapa da Jornada
Do primeiro contato até assinante fiel.

**Etapas:**
1. Descoberta (ad, busca orgânica)
2. Interesse (quiz, análise)
3. Consideração (resultado, comparação)
4. Decisão (oferta R$ 19/mês)
5. Onboarding (dashboard, rotina)
6. Uso regular (checklist diário)
7. Advocacy (indica para amigas)
