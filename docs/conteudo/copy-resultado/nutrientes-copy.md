## ✏️ Regras de Copy para Nutrientes

1. **Primeiro o porquê, depois o ativo** — "Seu problema é X. Esse nutriente age em X assim."
2. **Prazo honesto** — Sempre incluir prazo realista. Sem over-promise.
3. **Sentimento antes de resultado** — "O que você vai sentir" cria adesão emocional
4. **Mecanismo simplificado** — Uma frase de ciência para cada ativo aumenta a credibilidade
5. **Nunca listar ingredientes** — A usuária não compra ingrediente, compra transformação
