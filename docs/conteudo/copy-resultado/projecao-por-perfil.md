## ✏️ Regras de Copy para Projeção

1. **Marcos com prazo real** — D7, D14, D20, D28, D45, D60. Nunca "em algumas semanas"
2. **Marco D7 invisível é válido** — "acontece mas você não vê" cria credibilidade científica
3. **Nota científica ao final** — referência real sem ser acadêmica; humaniza o claim
4. **Nunca prometer eliminação** — "reduz", "diminui", "melhora", "começa a"
5. **Frase de ancoragem fecha o loop emocional** — sempre terminar com perspectiva de longo prazo
