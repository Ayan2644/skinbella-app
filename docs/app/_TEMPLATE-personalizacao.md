### Nível 2: Rotina Personalizada
[VOCÊ PREENCHE: Como a rotina é personalizada?]

**A rotina muda baseado em:**
- [ ] Tipo de pele (seca → mais hidratação)
- [ ] Idade (30+ → anti-aging, 40+ → firming)
- [ ] Problemas principais (acne → produtos específicos)
- [ ] Sensibilidade (pele sensível → produtos mais suaves)

**Exemplos de Rotinas:**

**Exemplo 1: Mulher 28 anos, pele oleosa, acne**
- Manhã:
  1. Limpeza (gel purificante)
  2. Tônico (ácido salicílico)
  3. Serum (niacinamida)
  4. Hidratante (oil-free)
  5. Protetor solar (toque seco)
- Noite:
  1. Limpeza dupla
  2. Tônico
  3. Serum (retinol 2x/semana)
  4. Hidratante leve

**Exemplo 2: Mulher 45 anos, pele seca, rugas**
- Manhã:
  1. Limpeza (cremosa)
  2. Tônico (hidratante)
  3. Serum (vitamina C)
  4. Hidratante (rico)
  5. Protetor solar (hidratante)
- Noite:
  1. Limpeza
  2. Tônico
  3. Serum (retinol 4x/semana)
  4. Creme anti-idade (peptídeos)

### Nível 5: Ajustes ao Longo do Tempo
[VOCÊ PREENCHE: A rotina evolui?]

**Após X semanas/meses:**
- [ ] Usuária faz novo quiz (reavaliar pele)
- [ ] Rotina é ajustada automaticamente
- [ ] Produtos são trocados (se pele melhorou, menos tratamento agressivo)
- [ ] Progresso é mostrado (comparação antes/depois)

**Gatilhos para Mudança de Rotina:**
- [ ] Após 4 semanas (primeira reavaliação)
- [ ] Após 8 semanas (segunda reavaliação)
- [ ] Mudança de estação (verão → mais proteção solar, inverno → mais hidratação)
- [ ] Usuária reporta novo problema (ex: começou acne hormonal)
