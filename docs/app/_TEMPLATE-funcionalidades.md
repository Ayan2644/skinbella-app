# FUNCIONALIDADES DESEJADAS - SKINBELLA APP

### ✅ MUST HAVE (Bloqueia lançamento se não tiver)

#### 1. Quiz de Análise de Pele
[VOCÊ PREENCHE: Como deve funcionar?]

- [ ] Tipos de pergunta: múltipla escolha? Escala? Imagens?

- [ ] Pode salvar progresso e continuar depois?

- [ ] Mostra antes/depois projetado?

#### 4. Dashboard / Tela Inicial (Today)
[VOCÊ PREENCHE: O que a pessoa vê ao abrir o app?]

- [ ] Streak counter (quantos dias seguidos)
- [ ] Progresso visual (scores melhorando ao longo do tempo?)
- [ ] Mensagem motivacional
- [ ] Acesso rápido para rotina

#### 5. Rotina Personalizada
[VOCÊ PREENCHE: Como funciona a rotina?]
- [ ] Quantas etapas? (limpeza, tônico, serum, hidratante, protetor solar?)
- [ ] É fixa ou muda ao longo do tempo?

#### 6. Biblioteca de Conteúdo (FAQ + Artigos)
[VOCÊ PREENCHE: Que tipo de conteúdo?]

#### 7. Sistema de Assinatura
[VOCÊ PREENCHE: Como funciona a assinatura?]
