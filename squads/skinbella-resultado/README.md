## O que este squad faz

Mapeia, analisa e otimiza a tela de resultado para conversão:
- Mapa de seções da ResultScreen com objetivos de conversão
- Hipóteses de A/B test priorizadas por impacto
- Variações do Sticky CTA por perfil
- Copy de urgência e escassez ético (apenas o que é real)
- Gap analysis da tela atual vs. estrutura ideal

**Output:** Documentos `.md` em `docs/conteudo/resultado/`
**Nunca:** Cria urgência falsa, gera código ou instrui implementação
