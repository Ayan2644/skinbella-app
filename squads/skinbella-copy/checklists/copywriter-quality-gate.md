# copywriter-quality-gate

Quality gate para outputs do agente **Val (copywriter)**.
Executar antes de marcar qualquer documento como ✅ aprovado.

---

## Checklist — Copy de Resultado

### 1. Tom de Voz
- [ ] Consulted `refs-tom-de-voz.md` antes de escrever
- [ ] Tom: amiga expert, não médico, não influencer exagerada
- [ ] Nenhuma linguagem de medo, vergonha ou urgência falsa
- [ ] Sem exagero: "radical", "milagre", "transformação total"

### 2. Personalização por Perfil
- [ ] Copy personalizado para CADA um dos 8 perfis
- [ ] "Por que para você" é específico, não genérico
- [ ] Dor do perfil refletida na headline ou lead

### 3. Claims
- [ ] Todos os claims passaram pelo `*validar-claims` do cientista
- [ ] Prazos alinhados com o que a ciência suporta
- [ ] Nenhum claim inventado sem base no documento de referência

### 4. Persona
- [ ] Copy passou pelo filtro: "A Marina entenderia e se identificaria?"
- [ ] Linguagem do dia a dia (não técnica demais)
- [ ] Extensão adequada — nenhum parágrafo > 80 palavras

### 5. CTA e Urgência
- [ ] Urgência é REAL e documentada (não criada para converter)
- [ ] CTA claro e sem ambiguidade
- [ ] Nenhum "últimas vagas" ou "só hoje" sem base real

### 6. Consistência de Preço
- [ ] ⚠️ Preço verificado — R$19/mês ✅ confirmado RESOLVIDO antes de publicar?
- [ ] Preço correto usado em todos os documentos de oferta

### 7. Formato
- [ ] Output é documento .md (nunca código)
- [ ] Estrutura clara por seção
- [ ] Notas de copy incluídas onde necessário
