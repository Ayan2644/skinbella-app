## Uso pelo Agente copywriter (Val)

Ao escrever copy de nutrientes ou oferta:
1. Carregar a lista de ativos com mecanismos
2. Usar linguagem acessível — não técnica demais
3. "Por que para você" parte dos mecanismos documentados aqui
4. Claims seguem o que está validado pelo cientista

## Nota

Claims devem ser validados pelo **cientista (Dra. Lara)** antes de publicar.
Se houver dúvida sobre um claim, escalar para `*validar-claims`.
